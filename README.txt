About
=====
It's everyone's favorite game from 2014! If you haven't played the game before, it's pretty simple. You control the bird who can't hit the pipes. It's the avian version of "the floor is lava".

Controls
========
A - Jump
SELECT - Restart
START - Start the game

Score
=====
Your score is determine by the number of pipes you successfully fly through. Try to beat your high score!
